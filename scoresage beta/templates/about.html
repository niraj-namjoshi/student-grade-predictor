Summary of ScoreSage
 The project undertook the development of a cutting-edge student grade predictor utilizing state-of-the-art machine learning techniques, particularly decision trees, using the powerful scikit-learn package.
The data preprocessing involved was carefully curated to deal with skewed dataset, and the model was trained using the most relevant data.
To connect the model with the aesthetically pleasing and user-friendly frontend, an advanced web framework, Flask, was deployed.
The project's data source was the prestigious UCI student repository, and it aimed to accurately predict student grades based on various parameters.
